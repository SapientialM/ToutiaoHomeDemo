## 2. 项目结构

## 3. 环境搭建

### 3.1 前置要求
- Android Studio 2025.1.1 或更高版本
- JDK 17+
- Android SDK API 36+（target），最低 API 26
- Git

### 3.2 克隆与构建
```bash
git clone <repo-url>
cd ToutiaoFeedDemo
./gradlew assembleDebug
# APK 输出: app/build/outputs/apk/debug/app-debug.apk
```

### 3.3 日志调试
```bash
# 查看应用日志（Timber 标签）
adb logcat -s Timber:D Toutiao:D MockInterceptor:D
```

### 3.4 Mock 数据
Demo 阶段使用 `MockInterceptor` 拦截所有 Retrofit 请求，无需真实后端。`MockInterceptor` 拦截 `https://api.example.com/` 开头的 URL，直接返回预置的 JSON 响应（8 条新闻，4 种卡片类型）。

### 3.5 常见问题
| 问题 | 解决方案 |
|------|----------|
| Gradle 同步失败 | 检查网络，已配置阿里云镜像源 |
| Compose 编译器版本不匹配 | 检查 BOM 版本与 Kotlin 版本兼容性 |
| Hilt 注解处理报错 | 项目使用 KSP（非 kapt），检查 KSP 配置 |
| 真机卡 Loading | MockInterceptor 未生效，检查 logcat 确认拦截日志 |

## 5. API 接口文档

### 5.1 获取新闻列表
```
GET /api/news/feed
```

**请求参数：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| channel | String | 是 | 频道 ID |
| page | Int | 是 | 页码，从 1 开始 |
| size | Int | 否 | 每页条数，默认 20 |

**响应示例：**
```json
{
  "code": 0,
  "data": {
    "list": [...],
    "hasMore": true
  }
}
```

**错误码：**
| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| -1 | 参数错误 |
| -2 | 服务器内部错误 |
